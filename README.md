# CodeAlpha_Unemployment_Analysis
# Unemployment Analysis using Python 📊

## Project Overview
This project analyzes unemployment trends in India using Python data analysis techniques.
The goal is to understand unemployment patterns and visualize the impact of COVID-19.

## Dataset
Dataset contains unemployment statistics across different states in India.

Dataset Source:
https://www.kaggle.com/datasets/gokulrajkmv/unemployment-in-india

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Project Steps
1. Data Cleaning
2. Exploratory Data Analysis
3. Visualization of unemployment trends
4. State-wise unemployment comparison

## Insights
- Unemployment rates increased during the COVID-19 period.
- Certain states experienced higher unemployment levels compared to others.
